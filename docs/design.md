# Design

## CLI Commands (Idea)

### View active connections
kmesh-debug connections

### View traffic for a specific service
kmesh-debug traffic service-a

### Watch live updates
kmesh-debug watch

## Features

- Real-time updates  
- Simple terminal output  
- Filter by service  
- Basic metrics display  

## Future Improvements

- Integration with Prometheus  
- Export to Grafana dashboards
